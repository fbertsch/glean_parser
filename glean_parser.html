glean_parser.parser.parse_metrics(filepaths)[source]¶

Parse one or more metrics.yaml files, returning a tree of metrics.Metric instances.

The result is a generator over any errors. If there are no errors, the actual metrics can be obtained from result.value. For example:

result = metrics.parse_metrics(filepaths)
for err in result:
    print(err)
all_metrics = result.value

The result value is a dictionary of category names to categories, where each category is a dictionary from metric name to metrics.Metric instances.
